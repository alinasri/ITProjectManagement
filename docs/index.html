How This App Works

A self-paced tutorial that walks through the architecture, database, and code of the IT Project Management app — written for someone learning React, Node, and Express through the app they actually built.

New to React/Node/Express? Every page below first explains the general concept in plain English (e.g. "what is a hook?", "what is middleware?"), then shows exactly how this app uses it, with real code from the repo. Read them in order — later pages build on earlier ones.
